# 🎬 VideoClub

Proyecto desarrollado en Java que permite gestionar un videoclub.

## 📌 Funcionalidades
- Gestión de clientes
- Gestión de películas
- Lectura y escritura en archivos .txt

## 🛠️ Tecnologías
- Java
- NetBeans

## ▶️ Cómo ejecutar
1. Abrir en NetBeans
2. Ejecutar el proyecto
